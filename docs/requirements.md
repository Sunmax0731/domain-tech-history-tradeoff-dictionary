# 要件定義

## 背景

AdobePlugin、AssetPipeline、VSCodeExtension、IoT、WindowsApp、WebApp、Gameなど各領域で使う技術を辞書的に記載する。歴史、登場背景、関連技術、代替技術、採用時のトレードオフを読み、タグで絞り込みや並び替えができる。

## 課題

技術名だけを調べても、なぜ使われるのか、何と比較すべきか、どの領域で重要かが分かりにくい。

## ユーザー価値

- 代表データを入力し、必須不足、警告、正常完了を同じ評価軸で確認できる。
- closed alpha 段階でも手順、責務、QCDS、release evidence を追える。
- アイデア棚卸しの領域分類と連動し、技術の意味だけでなく採用判断と関連領域まで辞書として引ける。

## MVP外では終わらせない範囲

- MVP後の UI/UX polish、責務分割、security/privacy、traceability、docs ZIP、release evidence を含める。
- 手動テストはユーザーが実施できる粒度で残す。
